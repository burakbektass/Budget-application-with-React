# React Router Budget App

Bu proje, React ve React Router kullanılarak geliştirilmiş modern bir bütçe yönetim uygulamasıdır. Kullanıcılar gelir-gider takibi yapabilir ve bütçelerini etkili bir şekilde yönetebilirler.

## 🚀 Tech Stack

- **React 18** - UI geliştirme için modern JavaScript kütüphanesi
- **React Router v6** - Sayfa yönlendirme ve navigasyon
- **Vite** - Hızlı geliştirme ortamı ve build tool
- **React Toastify** - Bildirim yönetimi
- **HeroIcons** - Modern ve şık ikonlar
- **SWC** - Hızlı JavaScript/TypeScript derleyicisi

## 🛠️ Kurulum

### Gereksinimler

- Node.js (v14.0.0 veya üzeri)
- npm (v6.0.0 veya üzeri)

### Kurulum Adımları

1. Projeyi klonlayın
```bash
git clone [proje-url]
cd react-router-budget-app
```

2. Bağımlılıkları yükleyin
```bash
npm install
```

3. Geliştirme sunucusunu başlatın
```bash
npm run dev
```

4. Tarayıcınızda açın 
[http://localhost:5173]


## 📦 Build

Projeyi production için build etmek:

```bash
npm run build
```

Build edilmiş versiyonu önizlemek için:
```bash
npm run preview
```
